# E-commerce-website-
A full-featured e-commerce mobile application built with React Native. Features include user authentication, product catalog, shopping cart, payment integration with Stripe, and order tracking.  Technologies Used: html css javascript ,react 
Complete E-commerce Features:

1. Multi-Page Application:
Home page with hero banner and featured products
Products page with grid view and filtering
Product detail page with images, specs, and add-to-cart
Shopping cart page with quantity adjustment
Checkout page with multi-step process
Order confirmation page
User profile page
Wishlist functionality

2. Shopping Cart System:
Add/remove items
Update quantities
Real-time cart badge updates
Cart persistence (localStorage)
Empty cart handling
Clear cart functionality

3. Product Catalog:
8 sample products across 4 categories (Optics, Feeders, Books, Apparel)
Product images, descriptions, specifications
Pricing with discount displays
Stock indicators
Ratings system

4. Checkout Process:
Multi-step checkout (Details → Shipping → Payment)
Form validation
Shipping options (Standard/Express)
Tax calculation
Order summary
Order confirmation with order number

5. User Features:
Wishlist with add/remove functionality
Notifications system
Profile management
Order history (placeholder)
Saved addresses (placeholder)
Settings (placeholder)

6. UI/UX Features:
Dark/light mode toggle
Loading screen
Toast notifications
Responsive design for mobile
Smooth transitions between pages
Hover effects and animations
Visual feedback for all interactions

7. Product Categories:
Optics: Binoculars, scopes
Feeders: Bird feeders, smart feeders
Books: Field guides, journals
Apparel: Jackets, hats, accessories

8. Birding-Specific Features:
Daily birding tips
Category-specific filtering
Bird-friendly product descriptions
Educational content integrated

9. Technical Features:
LocalStorage for data persistence
State management for cart/wishlist
Form validation
Image optimization
Performance optimization
